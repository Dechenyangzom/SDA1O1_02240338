## Pika16's Store ##

## Overview

This repository contains the HTML code for a simple e-commerce landing page titled "Pika16's Store." The page showcases a selection of featured products with images, descriptions, and prices.

## Table of Contents

- [Getting Started](#getting-started)
- [File Structure](#file-structure)
- [Guided Practical Exercises](#guided-practical-exercises)


## Getting Started

To view the landing page, you can either clone this repository or download the HTML file and open it in a web browser.

### Prerequisites

- A web browser (e.g., Chrome, Firefox, Safari)
- Basic understanding of HTML and CSS

## Guided Practical exercises

1.Modify Product Information:
    - Open index.html in a text editor.
    - Change the name and price of one of the products. For example, update the "Gaming Headphone with Mic" to "Gaming Headset" and change the price to "Nu.6000".

2.Add a New Product:
    -In index.html, locate the section with the class product-grid.
    -Copy one of the existing product cards (the entire <div class="product-card">...</div> block).
    -Paste it within the product-grid div.
    -Change the image source, product name, and price to reflect a new product

3.Style the Page:
    -Open landing_page.css in a text editor.
    -Experiment with changing the background color of the header by adding or modifying the CSS rule.
